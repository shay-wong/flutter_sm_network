# sm_network


| [English](README.md) | [简体中文](README.zh.md) |
| ------------------ | ----------------------- |

一个 Flutter 网络请求库，提供了一些常用的功能：

- 📱 支持 Dio 网络请求
- 📦 支持 Json 解析
- 🎉 支持打印日志到控制台
- 💥 支持打印错误信息到控制台
- *(剩余功能还在开发中...)*


## 特性

## 安装

运行下面的命令:

Dart:

```
dart pub add sm_network
```

Flutter:

```
flutter pub add sm_network
```

或者在 `pubspec.yaml` 文件中添加依赖并运行`dart pub get`：

```
dependencies:
  sm_network: ^0.0.1
```
