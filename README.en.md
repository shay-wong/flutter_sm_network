## Installing

Depend on it

Run this command:

With Dart:

```
dart pub add flutter_sm_network
```

With Flutter:

```
flutter pub add flutter_sm_network
```

Or add the dependency to your `pubspec.yaml` file and run `dart pub get`:

```
dependencies:
  flutter_sm_network: ^0.0.1
```
